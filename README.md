# Installation
0. Get TCC (`doas pkg_add tcc` or your respective distro's or OS's package manager)
1. Download the file
2. `tcc FILENAME -o OUTPUT_FILENAME`
# Usage
`./OUTPUT_FILENAME`
can also put in your `/bin/` or whatever to make it a simple bash command
